# Pete Thinkful Portfolio

## Overview
This is a static HTML/CSS portfolio website for "Pete Thinkful", an artist. The project is a simple, responsive portfolio template that showcases artwork and provides contact information.

## Project Type
- **Type**: Static HTML/CSS Website
- **Languages**: HTML, CSS
- **Build System**: None (static files)
- **Server**: Python HTTP Server on port 5000

## Project Structure
```
.
├── index.html          # Main portfolio page
├── style.css           # Styling and layout
├── images/             # Portfolio images
│   ├── pete-thinkful.png
│   ├── abstract-red.png
│   ├── spiral-zany.png
│   └── melted-rainbow.png
└── README.md           # Project documentation
```

## Features
- Responsive layout design
- Portfolio gallery with artwork samples
- About section with artist bio
- Contact section with social media links
- Custom fonts (Playfair Display, Source Sans Pro)
- Mobile-friendly design

## Setup
The site is served using Python's built-in HTTP server on port 5000. No build process or dependencies are required.
